## **How to install and run locally ?**

```
$ git clone https://github.com/NikhilNaroliya/Study-Help.git
$ cd  Study-Help
$ npm install
$ npm start
```
## **Features -**

 - SUBJECT LISTING PAGE
 - MARK AS READ PAGE
 - FAVOURITE PAGE
 - NOTES TAKING PAGE
 - CHAT BOT
 - PROFILE SECTION
 
 - Authentication:
 - google authentication
 - Fully Responsive

## **Built with -**

- React JS
- React Context API + useReducer
- React Router v6
- Google drive api
